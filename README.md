# Alpha Gobang Zero
基于强化学习的五子棋机器人。

## 快速开始
1. 创建虚拟环境并安装依赖包:
    
    ```shell
    conda create -n Alpha_Gobang_Zero python=3.8
    conda activate Alpha_Gobang_Zero
    pip install -r requirements.txt
    ```
    
2. 开始游戏:
    
    ```shell
    python Alpha_Gobang_Zero.py
    ```
    
    

## 环境要求
1. 电脑配有**英伟达 GPU**
2. 安装与电脑 GPU 相应的 Pytorch 版本
